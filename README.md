# url-shortener-node

## Getting started

1. Clone the repository
   
   ```bash
    git clone https://github.com/Smintfy/url-shortener-node.git
   ```
3. Install the depenencies
   
   ```bash
    cd url-shortener-backend
    npm install
   ```
4. Run the development server
   
    ```bash
    npm start
   ```
